# 📘 Assignment: HTML & CSS Fundamentals

## 🎯 Objective

Build your first web page using semantic HTML and CSS styling. You'll create a personal portfolio page that showcases your skills and projects, learning how HTML structures content and CSS brings it to life with colors, fonts, and layouts.

## 📝 Tasks

### 🛠️ Build Semantic HTML Structure

#### Description
Create a well-structured HTML page for a personal portfolio. Use semantic HTML elements like `<header>`, `<nav>`, `<main>`, `<section>`, and `<footer>` to organize your content logically. Include information about yourself, your skills, and featured projects.

#### Requirements
Completed program should:

- Include a `<header>` with a title or name
- Add a `<nav>` section with at least 3 navigation links
- Create a `<main>` section with at least 2 `<section>` elements (e.g., "About Me" and "Projects")
- Include a `<footer>` with copyright information
- Use semantic HTML elements (no `<div>` for major page sections)
- Have all HTML properly nested and indented


### 🛠️ Style with CSS

#### Description
Add visual styling to your portfolio page using CSS. Create an external stylesheet that defines colors, fonts, spacing, and layout to make your page visually appealing and easy to read.

#### Requirements
Completed program should:

- Use an external CSS file linked in the HTML
- Style at least 5 different elements (e.g., header, nav, sections, footer)
- Include custom colors (at least 3 distinct colors)
- Use at least 2 different fonts
- Add padding and margins for proper spacing
- Use flexbox or grid for at least one layout area
- Ensure text is readable with appropriate font sizes and contrast


### 🛠️ Add Responsive Design (Stretch Goal)

#### Description
Make your portfolio page responsive so it looks good on both desktop and mobile devices. Use CSS media queries to adjust the layout for smaller screens.

#### Requirements
Completed program should:

- Include a viewport meta tag in the HTML
- Use at least one media query for mobile devices (max-width: 768px)
- Adjust layout, font sizes, or navigation for mobile screens
- Test and verify appearance on different screen sizes
